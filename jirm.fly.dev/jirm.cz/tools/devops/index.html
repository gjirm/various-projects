Ansible

Ansible is a radically simple IT automation platform from RedHat that makes applications and systems easier to deploy and maintain. Automate everything from code deployment to network configuration to cloud management using SSH, with no agents to install on remote systems.

Terraform

Terraform is an open-source infrastructure as code software tool from HashiCorp that provides a consistent CLI workflow to manage hundreds of cloud services. Terraform codifies cloud APIs into declarative configuration files.
